DAS 反向记录

设置反向记录可让 DApp 显示你喜欢的 .bit 账户而不是冗长的区块链地址，同时让转账变得更加安全

应用集成
面向开发者

正向记录

一个 DAS 账户可以关联多种类的多个记录

将多个数据与 satoshi.bit 关联，应用（如钱包）可以通过查询 satoshi.bit 获得这些数据。

反向记录

一个地址只能设置一条反向记录

将一个地址指向一个特定的 DAS 账户，应用可以通过查询这个地址来获得这个 DAS 账户。

安全提升

DAS 反向记录带来了更安全的转账以及更安全的合约交互体验

当 Bob 需要向 Alice 发送加密货币，又或者需要和某个 DApp 的合约进行签名。

跨链

DAS 反向记录适用于所有链*

DAS 反向记录不仅仅适用于某条特定的链，它适用于所有链。任何链的地址都可以被设置为 DAS 反向记录。这意味着 DAS 反向记录可以整个 Web3.0 世界更加安全和友好。

*不同公链的支持的时间有所不同。

其实，它不仅仅适用于区块链

事实上，得益于 DAS 强大的兼容性和可拓展性，任何非对称密钥系统的公钥/地址都可以设置一个 DAS 反向记录，就像任何非对称密码系统的私钥都可以持有和管理 DAS 帐户一样。这也意味着 DAS 不仅能在区块链给我们带来更安全和友好的体验，更能在 Web3.0 中其他我们目前尚未想象到的场景大放异彩。
